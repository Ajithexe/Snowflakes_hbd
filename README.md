# ❄️ Winter Birthday Surprise

A cinematic winter-themed birthday surprise website created for my long-distance girlfriend. She loves snowflakes, so I built this experience for her using snowfall animations, frozen crack effects, glowing particles, and smooth cinematic transitions.

The project is designed to feel emotional, magical, and immersive — like stepping into a winter dream.

## 🧊 Concept

The experience begins with a single glowing snowflake slowly falling from the sky.

When the snowflake touches the ground:
- the screen cracks like frozen ice,
- light emerges through the cracks,
- and a magical snowstorm fills the screen,
revealing the birthday surprise.

This project was inspired by long-distance love and winter aesthetics.

> “She loves snowflakes, so I built this for her.” ❄️
